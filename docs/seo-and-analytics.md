# SEO & Analytics

## Meta Tags

All meta tags are rendered in `src/components/BaseHead.astro`.

### Props

```typescript
interface Props {
  title: string;
  description: string;
  image?: ImageMetadata;       // defaults to FallbackImage
  type?: 'website' | 'article'; // defaults to 'website'
  pubDate?: Date;
  updatedDate?: Date;
}
```

### Primary Meta Tags

- `<title>`, `<meta name="description">`, `<meta name="author">`
- Canonical URL derived from `Astro.url.pathname` + `Astro.site`

### Open Graph

- `og:type` (website or article), `og:url`, `og:title`, `og:description`
- `og:image` with `og:image:type` (jpeg), `og:image:width`, `og:image:height`, `og:image:alt`
- `og:site_name`, `og:locale` (en_US)
- OG image is processed via `getImage()` at 1200px width, JPEG quality 80

### Article-Specific (when `type === 'article'`)

- `article:published_time`, `article:modified_time`, `article:author`

### Twitter/X Cards

- `twitter:card` (summary_large_image)
- `twitter:site` and `twitter:creator` (@alexmorcillo82)
- `twitter:url`, `twitter:title`, `twitter:description`, `twitter:image`, `twitter:image:alt`

## JSON-LD Structured Data

### Homepage

`WebSite` + `Person` schema with `sameAs` linking to GitHub, X, and LinkedIn.

### Blog Posts (`src/layouts/BlogPost.astro`)

`BlogPosting` schema including:
- `headline`, `description`, `datePublished`, `dateModified`
- `author` and `publisher` as `Person`
- `articleSection` (category), `image`, `url`, `mainEntityOfPage`
- `isPartOf` linking to the Blog

### Breadcrumbs (`src/components/Breadcrumbs.astro`)

`BreadcrumbList` schema on category listings and blog posts.

### FAQ (`src/components/FAQSchema.astro`)

`FAQPage` schema — renders only a `<script type="application/ld+json">` tag, no visible HTML.

### Custom Events (`src/components/Analytics.astro`)

All custom tracking uses `plausible('EventName', { props: { ... } })` calls in a single inline script:

| Event | Props | Trigger |
|-------|-------|---------|
| `Theme Detected` | `theme` (dark/light) | Page load |
| `Theme Toggle` | `theme` (new value) | Theme toggle click |
| `Navigation Click` | `text`, `url`, `location` (header/footer) | Nav link click |
| `CTA Click` | `name` (`cookie_accept` / `contact_linkedin`), `url` | CTA button/link click |
| `Social Share` | `network`, `url` | Share button click |
| `Social Click` | `network`, `url` | Footer social icon click |
| `Scroll Depth` | `percentage` (25/50/75/100), `path` | Scroll milestones |

Page views are tracked automatically by Plausible; custom events track user intent (which links are clicked, scroll depth, etc.).

## RSS Feed

Custom implementation at `/rss.xml` (bypasses broken `@astrojs/rss` beta). Includes `<category>` per item and uses nested URLs.

Link tag in `BaseHead.astro`:
```html
<link rel="alternate" type="application/rss+xml" title="Alex Morcillo" href="/rss.xml" />
```

## Sitemap

Auto-generated by `@astrojs/sitemap` at `/sitemap-index.xml`. Referenced in `BaseHead.astro`:
```html
<link rel="sitemap" href="/sitemap-index.xml" />
```

## robots.txt

Located at `public/robots.txt`. Allows all crawlers and links to the sitemap.
